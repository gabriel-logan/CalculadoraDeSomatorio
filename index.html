Calculadora de somatorio

Exemplos de séries:

  • ((x ** k)) / (this.fatorial(k)) -----> e**x
  • 1/k -----> p serie divergente
  • 1/(k**2) -----> p serie convergente
  • ((-1)**k) / ( k ) -----> serie alternada convergente
  • (1/2)**k -----> serie geometrica convergente

Obs: Use ** para elevar o número a um expoente e, caso queira usar o fatorial, utilize this.fatorial(VALOR AQUI).
